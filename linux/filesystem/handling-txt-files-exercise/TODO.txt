Exercise: Linux Filesystem Commands 
-------------------------------------------------------------------------------

Führen Sie die folgenden Tätigkeiten auf der Linux Bash aus:

1) Wechseln Sie in das Verzeichnis "Downloads/"

2) Erstellen Sie darin ein Verzeichnis "data/"

3) Wechseln Sie in das Verzeichnis "data/" und erstellen Sie ein File mit dem 
    Namen "measurement.csv" und fügen Sie die folgenden Daten ein:
    number,timestamp,value
    1,2022-01-01 00:00:00,10.5
    2,2022-01-01 01:00:00,12.3
    3,2022-01-01 02:00:00,8.9
    4,2022-01-01 03:00:00,15.2
    5,2022-01-01 04:00:00,11.7
    6,2022-01-01 05:00:00,9.4
    7,2022-01-01 06:00:00,14.1
    8,2022-01-01 07:00:00,16.7
    9,2022-01-01 08:00:00,10.8
    10,2022-01-01 09:00:00,13.2

4) Überprüfen Sie die Größe des Files "measurement.csv"

5) Ändern Sie den Namen des Files auf "data.csv"

6) Geben Sie den Inhalt des Files auf der Console aus.

7) Löschen Sie zuerst das File "data.csv" und dann das Verzeichnis "data/".
