Exercise: Classes and Objects - LinkedStack
--------------------------------------------------------------------------------

Implement a class LinkedStack which can store a given number of int values. 


A) linked_stack.h
Declare the class LinkedStack with the following methods:
    bool isEmpty();
    void push(int value);
    int  pop();
    int  top();
    void print();
Also declare a struct Node holding an int value and a next_ptr.

B) linked_stack.cpp
Write the class implementation in a file called "linked_stack.cpp". 

Tip: Review the existing C implementation: 
    datastructures+algorithms/datastructures/stack/linked-stack

Tip: Run "test.cpp" to verify your implementations.
