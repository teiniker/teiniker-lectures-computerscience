Exercise: Classes and Objects - LCD
--------------------------------------------------------------------------------

Implement a class LCD which can be used to simulate a simple display.
We can create any number of displays (LCD objects) and use them.

A) lcd.h
Review the given lcd.h file to understand the structure of the 
LCD class.

B) lcd.cpp
Write the missing class implementation in a file called "lcd.cpp". 


Tip: Run "test.cpp" to verify your implementations.
